# Dwarka Toppers Website

Ready to upload directly to GitHub and publish with GitHub Pages.

## Included
- index.html
- assets/css/style.css
- assets/js/main.js
- assets/images/logo.svg
- robots.txt
- sitemap.xml

## Deploy
1. Create a GitHub repository.
2. Upload all files from this package.
3. Go to Settings > Pages.
4. Choose Deploy from a branch.
5. Select main branch and /root.
6. Connect custom domain dwarkatoppers.in after publishing.
